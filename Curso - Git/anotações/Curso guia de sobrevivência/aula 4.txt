HUB
---

Ferramenta que torna possível rodar várias funcionalidades da interface do github diretamente
no terminal do git.

hub create 'nome do repositório': Cria um repositório no github registrado no terminal.
_____________

Issues---

Trata-se de uma aba do github onde usuários das libs usam para comentar problemas que passaram com
a biblioteca, o que ajuda a outros usuários posteriores que possam vir a passar pelo mesmo problema.

Ao pesquisar no github, pode-se escrever o que se deseja pesquisar, como por exemplo:
"<p> hello world! </p>"
Assim, o github irá mostrar resultado de repositórios que correspondem ao texto dentro das aspas.
